# Table of Contents

> [!NOTE]
> This document provides an organized list of cybersecurity laboratory exercises covering core security operations, compliance, incident response, vulnerability management, and security tools.

## 📚 Laboratory Index

| Lab | Title | Description |
|------|-------|-------------|
| **Lab 01** | Security Monitoring & Alert Analysis | Monitor security events, analyze alerts, and identify potential security incidents. |
| **Lab 02** | Vulnerability Assessment & Remediation | Perform vulnerability assessments, analyze findings, and document remediation activities. |
| **Lab 03** | Security Audits & Compliance Reviews | Conduct security audits and evaluate compliance against established security requirements. |
| **Lab 04** | Cybersecurity Incident Investigation | Investigate cybersecurity incidents, collect evidence, and document findings. |
| **Lab 05** | Security Policy & Documentation Management | Review, manage, and maintain security policies, procedures, and documentation. |
| **Lab 06** | Emerging Threats Research | Research emerging cybersecurity threats, attack techniques, and industry trends. |
| **Lab 07** | Endpoint, Network & Application Security | Assess and implement security controls for endpoints, networks, and applications. |
| **Lab 08** | Log Review & Anomaly Detection | Analyze system and security logs to detect anomalies and suspicious activities. |
| **Lab 09** | IT & Security Team Collaboration | Participate in collaborative security operations, communication, and cross-functional activities. |
| **Lab 10** | Security Awareness Training Activities | Support and participate in cybersecurity awareness and user education initiatives. |
| **Lab 11** | Security Tools Hands-On (Wireshark, Nmap, Burp Suite) | Perform practical exercises using industry-standard cybersecurity tools. |
| **Lab 12** | Frameworks & Standards (ISO 27001, NIST, OWASP) | Study and apply recognized cybersecurity frameworks, standards, and best practices. |

> [!TIP]
> Complete the laboratories sequentially to build foundational knowledge before progressing to advanced cybersecurity concepts and practical exercises.
